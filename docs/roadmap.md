# Roadmap / 产品路线图

This roadmap describes the public-facing direction of **The Beauty of Life / 故事坊**.  
It does not expose private source code, backend implementation, database design, private prompts, API keys, or user data.

这份路线图用于说明 **The Beauty of Life / 故事坊** 的公开产品方向。  
它不会公开完整源码、后端实现、数据库设计、私有 Prompt、API Key 或用户数据。

**Goal:** Build a trustworthy public-facing project page without exposing the commercial core.  
**目标：** 建立一个可信的公开展示仓库，同时保护商业核心代码和实现细节。

- Create the public GitHub showcase repository  
  创建公开 GitHub 展示仓库
- Add Chinese README and product introduction  
  添加中文 README 和产品介绍
- Add product screenshots  
  添加产品截图
- Add product deck PDF  
  添加产品演示 PDF
- Add product vision, privacy principles, design principles, and feature map  
  添加产品愿景、隐私原则、设计原则和功能地图
- Add online testing address  
  添加在线测试地址
- Add README cover image  
  添加 README 封面图

## Phase 2: Product Prototype / 第二阶段：产品原型完善

**Goal:** Improve the core AI interview and memory organization experience.  
**目标：** 完善 AI 访谈和回忆整理的核心体验。

- Improve AI-guided life story interview flow  
  优化 AI 引导式人生故事访谈流程
- Refine elder-friendly UI and interaction details  
  优化适老化界面和交互细节
- Improve voice input, recording upload, and text input experience  
  优化长按说话、录音上传和文字输入体验
- Improve topic progress and topic switching experience  
  优化主题进度和换题体验
- Improve memory library and story snippet presentation  
  优化回忆库和故事片段展示
- Improve biography generation quality  
  提升回忆录生成质量
- Improve PDF / Word export experience  
  优化 PDF / Word 导出体验

## Phase 3: Private Beta / 第三阶段：小范围内测

**Goal:** Test the product with real users in a controlled and privacy-aware way.  
**目标：** 在可控、重视隐私的前提下，邀请真实用户进行小范围测试。

- Invite early families and elderly users  
  邀请早期家庭用户和长辈用户
- Collect feedback on interview comfort and usability  
  收集访谈舒适度和易用性反馈
- Test whether elders can use the product without heavy assistance  
  测试长辈是否能在较少协助下使用产品
- Test memory quality and biography readability  
  测试回忆整理质量和回忆录可读性
- Improve privacy consent and account deletion flow  
  优化隐私同意和账号注销流程
- Prepare safe demo accounts and sample stories  
  准备安全的演示账号和示例故事

## Phase 4: Trust, Security, and Deployment / 第四阶段：信任、安全与部署

**Goal:** Prepare the product for broader external access.  
**目标：** 为更大范围的外部访问做好基础设施和信任准备。

- Bind a formal domain name  
  绑定正式域名
- Enable HTTPS  
  启用 HTTPS
- Harden server security and exposed ports  
  加固服务器安全和开放端口
- Ensure environment variables and API keys are not exposed  
  确保环境变量和 API Key 不暴露
- Improve production monitoring and deployment checklist  
  完善生产监控和上线检查清单
- Prepare a public demo guide  
  准备公开体验指南
- Clarify data storage, deletion, and family voice authorization boundaries  
  明确数据存储、删除和家人声音授权边界

## Phase 5: Commercial Product / 第五阶段：商业化产品

**Goal:** Explore sustainable commercialization while keeping user trust at the center.  
**目标：** 在坚持用户信任的前提下，探索可持续商业化。

- Define pricing model  
  明确定价模型
- Define target customer segments  
  明确目标客户群体
- Build reliable onboarding flow  
  建立稳定的新用户引导流程
- Improve family collaboration features  
  完善家庭协作功能
- Explore book-style export and printing services  
  探索书籍式导出和打印服务
- Prepare public launch materials  
  准备公开发布材料
- Decide whether to open-source selected non-core components  
  决定是否开源部分非核心组件

## Long-Term Vision / 长期愿景

The long-term vision of **The Beauty of Life / 故事坊** is to become a warm, privacy-first family memory product that helps ordinary people preserve life stories across generations.

**The Beauty of Life / 故事坊** 的长期愿景，是成为一个温暖、重视隐私的家庭记忆产品，帮助普通人把人生故事跨代保存下来。
